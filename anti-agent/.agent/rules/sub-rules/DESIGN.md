# DESIGN & UX RULES - Antigravity Kit

## 🎨 Color Palette (Forbidden: Purple/Violet)
- **Primary**: Sleek Dark (#121212), Deep Blue (#1A237E), Emerald Green (#2E7D32)
- **Accent**: Sunset Orange (#F57C00), Electric Cyan (#00E5FF)
- **Glassmorphism**: rgba(255, 255, 255, 0.1) with backdrop-filter: blur(10px)

## 🖋️ Typography
- **Core Font**: Inter, Roboto, Outfit (Google Fonts)
- **Hierarchy**: H1 (700 wt, 2.5rem), H2 (600 wt, 2rem)

## Component Standards
- **Shadows**: Soft, multi-layered shadows (box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1))
- **Radii**: 12px for cards, 8px for buttons.
- **Micro-animations**: Transition all 0.2s ease-in-out.
- **Accessibility**: Aria-labels on all interactive elements.
